## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

// SQL CODE, testing with postgres
select
b.id as brand_id,
b.name as brand_name,
o.id as outlet_id,
o.name as outlet_name,
o.address as outlet_address,
o.longitude as outlet_longitude,
o.latitude as outlet_latitude,
(select distinct count(id) from products
where o.brand_id = b.id
) as total_product
from brands as b
left outer join public.outlets o on b.id = o.brand_id
group by o.id, b.id
